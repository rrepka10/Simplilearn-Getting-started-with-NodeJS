# Getting-started-with-NodeJS
ms.simplilearn.com offers a NodeJS course called "Getting-started-with-NodeJS".  

This repo contains the sample code from the class with updated for the latest version
of NodeJS Mongose.  This repo also includes instructions on how to setup each
module.  
There are 5 sections:
```
express
rest
jwt
mysql
mongodb	- updated to use mongodb cloud 
	requires mongodb\.env with 
		DB_ADMIN=<your mongodb id>
		DB_PASSWORD=<your mongodb password>
```
